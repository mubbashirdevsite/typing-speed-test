# ⌨️ Typing Speed Test App

A modern **Typing Speed Test application** built with HTML, CSS, and JavaScript to measure WPM, CPM, and accuracy in real time.

---

## 🚀 Features

- Random paragraph generator (30–50 samples)
- Real-time WPM calculation
- CPM tracking
- Accuracy percentage
- 3 time modes (30s / 60s / 120s)
- Live character highlighting (correct / incorrect)
- Progress bar timer
- Best WPM stored in LocalStorage
- Restart test anytime
- Fully responsive UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API
